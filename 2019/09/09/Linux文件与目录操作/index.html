Linux文件与目录操作

发表于 2019-09-09 更新于 2019-09-11 分类于 Linux

目录处理

cd change directory 切换目录

pwd print working directory 显示当前目录

mkdir 新建目录

rmdir 删除空目录

文件处理

cp copy

默认情况下目的文件的所有者是命令操作者
没有参数时 cp 复制的是源文件而非连接文件

rm remove

目录名以 - 开头会造成误判，可以用 ./ 避免

mv move

多个源文件或目录要移动时，最后一个（即目标文件）一定是目录 废话

文件内容查阅

直接查看

  • cat concatenate 从第一行开始显示文件内容

  • tac cat 反写，功能也是

  • nl 添加行号显示

翻页查看

  • more
    Space ：向后翻一页
    Enter ：向下滚动一行
    /字符串 ：当前页向下查询，n 继续查询
    :f ：显示出文件名以及目前显示的行数
    b/ctrl+b ：回翻，仅限文件，FIFO无效
  • less
    Space/PageDown ：向后翻一页
    PageUp ：向前翻一页
    ?字符串 ：当前页向上查询，n 继续查询
    N ：反向继续查询

数据选取

head 取出前几行，默认显示十行

tail 取后面几行

od 非纯文本文件

文件时间处理

  • mtime modification time
    内容数据更改时更新，ls 默认值
  • ctime status time
    状态改变时更新，如权限与属性
  • atime access time
    内容被取用访问时更新

默认将三个时间更新为当前，若文件不存在创建新文件
