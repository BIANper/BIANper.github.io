内存地址比较用 equals
string 类型与 null 比较用 ==
string 类型与空字符串 “” 比较用 equals

<u>乱入</u>
```java
String str1 = null; //str引用为空   
String str2 = ""; //str引用为空串   
```
<u>相关</u>
如果 str1=null; 这个的写法错误⬇：
```java
if(str1.equals("")||str1==null){  
//如果str1没有值，则返回false  
}   
```
正确的写法是：
```java
if(str1==null||str1.equals("")){   
//先判断是不是对象，如果是，再判断是不是空字符串   
}
```
