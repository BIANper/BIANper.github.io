1. 子类给成员变量分配空间，在栈中默认初始化成0
2. 调用子类构造函数，因为继承，调用 super();
3. 调用父类构造函数，因为覆盖，执行子类方法
4. 执行父类构造函数其他语句
5. 父类构造函数结束弹栈
6. 子类对成员变量显示初始化(赋值)
7. 调用子类构造函数其他语句   

<!--more-->
**检验**
```java
class Fu{
    Fu()
    {
        this.getNum();
    }
    public void getNum()
    {
        System.out.println("父类");
    }
}
 
class Zi extends Fu{
    int num=4;
    public Zi()
    {
        super();
        System.out.println("子类构造函数中"+this.num);
    }
    public void getNum()
    {
        System.out.println("getNum方法"+this.num);
    }
}
 
public class learn{
    public static void main(String[] args)
    {
        new Zi();
    }
}
```
